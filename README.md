# 安卓面试题库
各位Developers 跳槽或者校招时，最重要、最烦恼的就是准备面试，为了方便大家准备面试，创建了Android面试题库。
将常见面试知识点、各大厂的社招/校招面试题、以及个人浅薄的开发经验，收集整理出来分享给大家。
欢迎大家加入，一起整理并维护这个题库，如果觉得有帮助👍，欢迎✨Star✨
